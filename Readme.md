# Joe Rocha's Portfolio
This is a conversion of `AdroitGraphics.com` to SSR with Node.js + Express + Next.js.

## Getting Started
Install dependencies.
```
$ npm install
# or
$ yarn install
```

## Usage
Feel free to replace npm with yarn.
```
// Dev (HMR enabled)
$ npm run dev

// Compile for production
$ npm run build

// Deploy
$ npm run start
```
